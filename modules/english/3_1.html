The Computer Keyboard

A keyboard is a piece of computer hardware that is used to input information into the computer or to interact with the computer in general. Since it is technically not a part of the computer, it is called a peripheral.

Most keyboards use the same layout as a typewriter for the number and letter keys. There are also additional keys which are used for various tasks such as moving the cursor (blinking light that shows where it is you are typing) around or sending different commands to the computer such as turning the volume up or down. All the different types of keys will be explained in the next unit.
