Important Keys

There are few keys that are important for everyday use. The ones we will focus on are enter, backspace, ctrl, alt, shift, and esc.

Start with the enter key. This can be used to get to a new line in a document. It can also be used to submit a form or go to a web address after typing it. Think of enter as a way of saying “I’m done, execute this command”.

The backspace key is important when typing. If you type a letter that you did not want to type, backspace will get rid of it for you.

The control key (labeled as ctrl) and the alternate key (labeled as alt) are often used for keyboard shortcuts. For example, if I am in a word document and I want my letters to be bold I can use my mouse to click the letter B at the top of the screen, or I can hold down the ctrl key and press the b key on my keyboard and it will have the same effect without needing to touch the mouse.

Shift is used in typing to slightly modify the item that is being typed. When you hold down the shift key and press a letter, instead of lowercase that number will be typed as capital. For other keys that show more than one symbol on them, the number keys for example, the lower one is typed when shift is not down, but the upper symbol is types when the user holds shift while pressing the key.

Lastly the escape key (labeled as esc) can be used if you need to get out of something. If you open a video and it takes up your entire screen and you can’t figure out how to exit, try using the escape key.
