What is a computer?

A computer is an electronic device that is used to create, process, store and retrieve data. We then use these data to formulate helpful information such as documents, music, images, videos, etc. In addition to creating helpful information, a computer is also useful to search all types of information, watch videos, send email, play video games, pay bills, etc.! [1] Figure 1 is an example of a desktop computer.

Hardware and Software

There are different types of computers. However, they all have two main things in common: Hardware and Software. The computer hardware refers to the physical parts of the computer (see Figure 2, left). Some examples of computer hardware include, the keyboard, the computer mouse, the monitor, the system unit, or any other computer peripheral that connects to the desktop computer. It also includes the parts inside the system unit, such as the CPU and the motherboard [1].

The computer software refers to instructions (or code) that aid the computer hardware what to do and how to do it. There are many types of computer software such as operating systems, web browsers, games, word processors, and more! The basic software of a computer consists of system software (see Figure 2, right) and application software [1].
