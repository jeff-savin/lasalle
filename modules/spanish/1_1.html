¿Qué es una computadora?

Una computadora es un dispositivo electrónico que se utiliza para crear, procesar, almacenar y recuperar datos. Luego usamos estos datos para formular información útil como documentos, música, imágenes, videos, etc. Además de crear información útil, una computadora es muy útil para buscar todo tipo de información, mirar videos, enviar correos electrónicos, jugar videojuegos, pagar facturas, ¡etc.! ¿Qué es hardware y software?

Hay diferentes tipos de computadoras. Sin embargo, todas tienen dos cosas principales en común: Hardware y Software. El hardware de la computadora se refiere a las partes físicas de la computadora (consulte la figura 2, a la izquierda). Algunos ejemplos de hardware de computadora incluyen el teclado, el mouse de la computadora, el monitor, la unidad del sistema o cualquier otro periférico de computadora que se conecte a la computadora de escritorio. También incluye las piezas dentro de la unidad del sistema, como la CPU y la placa base [1].

El software de la computadora se refiere a las instrucciones (o código) que ayudan al hardware de la computadora a saber qué hacer y cómo hacerlo. ¡Hay muchos tipos de software de computadora, como sistemas operativos, navegadores web, juegos, procesadores de texto y más! El software básico de una computadora consta del software del sistema (consulte la figura 2, a la derecha) y el software de aplicación [1].
