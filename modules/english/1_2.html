A desktop computer and its main parts

A desktop computer (see figure 1) is a computing device designed to fit on top of an office desk due to its size and power requirements. Its main parts consist of a system unit, a monitor, a keyboard, and a mouse [1, 3].

The system unit

The computer system unit (see figure 2) is a boxlike case that houses additional components of the computer, including the motherboard, central processing unit (CPU), power supply, hard disk drive, and optical disc drive [4, 5]. Other names for the system unit include the computer chassis, cabinet, box, tower, or simply case. Most people erroneously refer to the system unit as the CPU.

The computer monitor

The computer monitor (see figure 3) is an output device which displays information in pictorial form (images and text). It comprises of circuitry, a screen, a power supply, buttons to adjust screen settings, and casing that holds all these components [6, 7].

The computer keyboard

The computer keyboard (see figure 4) is an input device that is used to enter characters (e.g. letters, numbers and other symbols) into a computer system by pressing different buttons or keys [5, 8]. There are different types of keyboards (e.g., wired, wireless, mechanical, membrane, QWERTY, DVORAK, etc.), but most allow you to achieve the same basic tasks [9].

The computer mouse

The computer mouse (see figure 5) is an input device that controls a cursor on the computer screen to interact with files, folders, windows, and other software elements [10]. There are two main mouse types: optical and mechanical [5].

A laptop and its main parts

A laptop is a battery-powered computer that is portable and just about as powerful as a desktop computer [1]. Despite looking different, the laptop and the desktop computer use the same programs and perform similar tasks (see figure 6) [11].

The webcam

A webcam is a type of input device that can record videos and take pictures. It can also transmit video over the Internet in real time, which allows for video chat or video conferencing with someone else [12]. Laptops have a webcam above its screen (see figure 7) [13].

The display (or screen)

The display (or screen), like a desktop monitor, is an output device which displays information in pictorial form (images and text). It is built into the laptop’s lid (see figure 8) [14].

The keyboard

The keyboard is an input device that is used to enter characters (e.g., letters, numbers, and other symbols) into a laptop by pressing buttons or keys [5]. It is located beneath the laptop’s display, and unlike they keyboard on desktop computers, most laptop keyboards do not have a numeric keypad (see figure 9) [15].

The touchpad

The touchpad also called a trackpad is an input device that does the same thigs as a computer mouse [16]. It is a touch-sensitive pad that lets you control the pointer by making a drawing motion with your finger (see figure 10) [17].
