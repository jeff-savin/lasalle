What is a Computer?

A computer is an electronic device to create, process, store or retrieve different types of data and information (see Figure 1). A computer is used to create and edit documents, search information, watch videos, listen to music, send email, play games, and much more! All computers have two things in common: hardware and software

Hardware and Software

The hardware refers to any part of the computer with a physical structure (see Figure 2, left). For example, the keyboard, the mouse, the monitor, the system unit, or any other physical peripheral that connects to the computer. The software refers to any set of instructions (or code) that tells the computer what to do and how to do it. Examples of computer software include operating systems, web browsers, games, word processors, and more! The basic software of a computer consists of system software (see Figure 2, middle) and application software (see Figure 2, right)
