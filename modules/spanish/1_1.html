¿Qué es una computadora?

Una computadora es un dispositivo electrónico para crear, procesar, almacenar o recuperar diferentes tipos de datos y de información (consulte la Figura 1). Se utiliza para crear y editar documentos, buscar información, ver videos, escuchar música, enviar correos electrónicos, jugar juegos y mucho más. Todas las computadoras tienen dos cosas en común: hardware y software [1, 2].

¿Qué es Hardware y Software?

El hardware se refiere a cualquier parte de la computadora con una estructura física (consulte la Figura 2, izquierda). Por ejemplo, el teclado, el ratón, el monitor, la unidad del sistema o cualquier otro periférico físico que se conecte a la computadora. También incluye las partes internas de la computadora, como el disco duro o la placa base [1].

El software se refiere a cualquier conjunto de instrucciones (o código) que le dice a la computadora qué hacer y cómo hacerlo. Los ejemplos de software de computadora incluyen sistemas operativos, navegadores web, juegos, procesadores de texto y más. El software básico de un ordenador consta de software del sistema (consulte la Figura 2, medio) y software de aplicación (consulte la Figura 2, derecha) [1].
