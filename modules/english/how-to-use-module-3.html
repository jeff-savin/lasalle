How to use Module 3: The Computer Keyboard

 

By the end of this module:

1.       Students will be able to define a computer keyboard.

2.       Students will be able to identify the different areas of the computer keyboard.

3.       Students will be able to differentiate between a wired and a wireless computer keyboard.

4.       Students will be able to connect a wired and a wireless computer keyboard to a desktop computer.

5.       Students will be able to recognize the home row of the computer keyboard.

6.       Students will be able to demonstrate knowledge of basic computer keys.


Note to the teacher: This self-guided manual provides some ideas on how to present the content for each of the lessons in module 3. It is by no means the only way to present the content to students.


Lesson 1: The Computer Keyboard

By the end of this lesson:

1.       Students will be able to will be able to define a computer keyboard.

 

Note to the teacher: A helpful exercise for this lesson is to instruct students to place both of their hands on the computer keyboard. It is ok if they place their fingers on the incorrect keys for now. Let them press on any of the keys. As they do this, ask them if they notice anything different about the keys (i.e., location, size, feel, etc.). In addition to pressing on each of the keys, ask them to take some time look at the keyboard and check the names, numbers, and characters.


Lesson 2: Areas of the Keyboard

By the end of this lesson:

1.       Students will be able to differentiate between a wired and a wireless computer keyboard.

 

Note to the teacher: In this lesson, place emphasis on the alphanumerical and numerical keys. Tell students the differences between them and, if possible, have them press all these keys on a MS Word document. It is helpful to have several keyboard exercises for this lesson and future lessons.


Lesson 3: Wired and Wireless Keyboards

By the end of this lesson:

1.       Students will be able to differentiate between a wired and a wireless computer keyboard.

 

Note to the teacher: It can be very helpful to bring a wired and a wireless keyboard to class. Pass them around for students to take a closer look. This will help students see the main difference between these two.

 As the keyboard gets passed around, point out some of its main keys such as the enter key, the spacebar, and the shift key.


Lesson 4: Connecting a Keyboard to a Computer

By the end of this lesson:

1.       Students will be able to connect a wired and a wireless computer keyboard to a desktop computer.

 

Note to the teacher: For this lesson, place emphasis on the USB port and the USB receiver. If possible, demonstrate how to connect a wired and wireless keyboard to a desktop computer and to a laptop.


Lesson 5: Identify the Home Row

By the end of this lesson:

1.       Students will be able to recognize the home row of the computer keyboard.

 

Note to the teacher: It is advisable to place emphasis on keyboard exercises for this lesson. Focus on more exercises than on the theory. If possible, take the time to show each student how to place their hands on the keyboard (after showing a video on how to do this on the class projector).


Lesson 6: Important Keys

By the end of this lesson:

1.       Students will be able to demonstrate knowledge of basic computer keys.

 

Note to the teacher: Just like in the previous lesson, it is very helpful to have several keyboard exercises ready for students. The important keys are usually covered in most keyboard exercises. As students perform these exercises, walk around the classroom and check that students have both of their hands placed on the keyboard correctly. For those who do not have the correct position, remind them how to do so.
