Module 3: The Computer Keyboard

Lesson 3: Wired and Wireless Keyboards

1. Students will be able to differentiate between a wired and a wireless computer keyboard.


Video



A Wired Keyboard

A wired keyboard is a keyboard that requires a wire to connect to the computer. The wire is usually permanently attached to the keyboard.

A Wireless Keyboard

A wireless keyboard doesn’t need any wires to connect to the computer. It does need a special receiver (the small item next to the keyboard) which plugs into the computer and receives the signals from the keyboard. Wireless keyboards also need batteries installed since they don’t receive any power from the computer.
