A desktop computer and its main parts

A desktop computer (see figure 1) is a computing device designed to fit on top of an office desk due to its size and power requirements. Its main parts consist of a system unit, a monitor, a keyboard, and a mouse [1, 3].

The system unit

The computer system unit (see figure 2) is a boxlike case that houses additional components of the computer, including the motherboard, central processing unit (CPU), power supply, hard disk drive, and optical disc drive [4, 5]. Other names for the system unit include the computer chassis, cabinet, box, tower, or simply case. Most people erroneously refer to the system unit as the CPU. /p>

The computer monitor

The computer monitor (see figure 3) is an output device which displays information in pictorial form (images and text). It comprises of circuitry, a screen, a power supply, buttons to adjust screen settings, and casing that holds all these components [6, 7].

The computer keyboard

The computer keyboard (see figure 4) is an input device that is used to enter characters (e.g. letters, numbers and other symbols) into a computer system by pressing different buttons or keys [5, 8]. There are different types of keyboards (e.g., wired, wireless, mechanical, membrane, QWERTY, DVORAK, etc.), but most allow you to achieve the same basic tasks [9].

The computer mouse

The computer mouse (see figure 5) is an input device that controls a cursor on the computer screen to interact with files, folders, windows, and other software elements [10]. There are two main mouse types: optical and mechanical [5].
